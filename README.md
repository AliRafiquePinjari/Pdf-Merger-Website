# PDFusion - PDF Merging Website

PDFusion is a simple web application that allows users to merge multiple PDF files into a single PDF document. This project utilizes Node.js for the backend logic and Bootstrap for the frontend design.

## Screenshot 
![image](https://github.com/AliRafiquePinjari/Pdf-Merger-Website/assets/105441038/d76a13e8-8d56-49b9-9747-77198f9b752f)

## Features

- **Merge PDF Files**: Users can upload multiple PDF files and merge them into a single PDF document.
- **Simple Interface**: The website provides a user-friendly interface for easy PDF merging.
- **Responsive Design**: Bootstrap is used to ensure the website looks great on various devices and screen sizes.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

git clone https://github.com/AliRafiquePinjari/Pdf-Merger-Website.git

2. Install dependencies:
  cd pdfusion
  npm install

3. Start the server:


4. Open your web browser and visit `http://localhost:3000` to access the PDFusion website.

## Usage

1. Open the PDFusion website in your web browser.
2. Click on the "Choose Files" button and select the PDF files you want to merge.
3. Click the "Merge" button to combine the selected PDF files.
4. Once the merging process is complete, the merged PDF will be available for download.
